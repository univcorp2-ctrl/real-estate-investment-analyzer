# Kantei-san 投資条件・判定基準詳細

## 0. このドキュメントの読み方

この資料は、Kantei-san がどのような条件で物件を評価するかを、初心者でも追えるように細かく整理したものです。

不動産投資の判定では、1つの数字だけで良し悪しを決めません。

たとえば、表面利回り12%の物件でも、次のような場合は危険です。

- 空室が多い
- 修繕費が大きい
- ローン返済が重い
- 土地値が低い
- 売却しにくいエリア
- 災害リスクが高い
- 再建築できない
- 賃料が相場より高く、下落しそう

Kantei-san は、これを6つの軸で点数化します。

## 1. 総合スコア

### 1.1 点数構成

```text
総合スコア = 収益性 + 立地 + 融資条件 + リスク耐性 + 価格妥当性 + 出口戦略
```

各軸は0〜5点です。

```text
最大点 = 6軸 × 5点 = 30点
```

### 1.2 ランク

| 総点 | 基本ランク | 初心者向け意味 |
|---:|---|---|
| 22〜30 | S | かなり良い候補。ただし詳細確認は必要 |
| 20〜21 | A | 良い候補。確認ポイントを潰して検討 |
| 15〜19 | B | 条件次第。価格や融資条件を調整したい |
| 12〜14 | C | 慎重検討。初心者は無理しない方がよい |
| 8〜11 | D | 原則見送り。リスクが目立つ |
| 0〜7 | E | 見送り。投資対象として危険度が高い |

### 1.3 なぜ総点だけで決めないのか

総点だけで判断すると、次のような危険があります。

```text
収益性 5点
立地 5点
融資条件 0点
リスク耐性 5点
価格妥当性 5点
出口戦略 5点
合計25点
```

合計は高いですが、融資条件が0点なら、返済できない可能性があります。

そのため、Kantei-san は **ゲート判定** を入れます。

## 2. ゲート判定

ゲート判定とは、重大リスクがある場合に、総点が高くてもランク上限を下げる仕組みです。

### 2.1 ゲート条件

| 条件 | 意味 | ランク上限 |
|---|---|---|
| DSCR < 1.0 | 物件収入だけでは返済を賄えない | D |
| 月間CF < 0 | 毎月赤字 | D |
| 表面利回り < 5% | 収益力が低い | C |
| 損益分岐稼働率 > 100% | 満室でも赤字になりやすい | D |
| 土地値カバー率 < 40% かつ 法定耐用年数超過 | 出口が弱い | D |
| ハザード重大該当 | 災害・保険・融資・出口に影響 | CまたはD |
| 再建築不可 | 出口・融資に大きく影響 | CまたはD |
| 重要データ不足 | 判定の信頼度が低い | AまたはC |

### 2.2 例

#### 例1: 点数は高いがCF赤字

```text
総点: 24点
基本ランク: S
月間CF: -12,000円
最終ランク: D
```

理由:

- 毎月赤字なので、初心者にとって危険
- 給与から補填する前提になっている
- 空室や修繕が来るとさらに悪化する

#### 例2: 総点は22点だがデータ不足

```text
総点: 22点
基本ランク: S
土地面積未入力
路線価未入力
近隣賃料未確認
最終ランク: A
```

理由:

- 点数は高いが、出口と価格妥当性の根拠が弱い
- 詳細確認前にSとは言えない

## 3. 収益性スコア

### 3.1 見る指標

収益性では主に以下を見ます。

- 表面利回り
- NOI利回り
- 月間CF
- DSCR

### 3.2 表面利回りによる基礎点

| 表面利回り | 点数 | 初心者向け意味 |
|---:|---:|---|
| 12%以上 | 5 | 収益性は高い。ただし高利回りにはリスクも多い |
| 10%以上 | 4 | 比較的高収益 |
| 8%以上 | 3 | 検討ライン |
| 6%以上 | 2 | 低め。融資条件や出口次第 |
| 4%以上 | 1 | 収益性は弱い |
| 4%未満 | 0 | 投資として厳しい可能性が高い |

### 3.3 補正

| 条件 | 補正 |
|---|---:|
| 月間CFがマイナス | -1点 |
| DSCRが1.0未満 | -1点 |
| NOIがマイナス | 0点固定 |
| 賃料が相場より20%以上高い | -1点 |
| 現況空室で想定賃料のみ | -0.5〜-1点 |

### 3.4 初心者向けの注意

表面利回りが高い物件ほど、以下のような問題が隠れていることがあります。

- 空室が多い
- エリア需要が弱い
- 建物が古い
- 修繕費が大きい
- 再建築不可
- 土地値が低い
- 売却しにくい

そのため、Kantei-san では表面利回りだけではS判定にしません。

## 4. 立地スコア

### 4.1 見る指標

- 駅徒歩
- 最寄駅の乗降客数
- 人口トレンド
- 将来人口
- 用途地域
- 人口集中地区かどうか
- 賃貸需要
- 売却時の流動性

### 4.2 駅徒歩

| 駅徒歩 | 点数目安 |
|---:|---:|
| 5分以内 | +2.0 |
| 10分以内 | +1.5 |
| 15分以内 | +1.0 |
| 20分以内 | +0.5 |
| 20分超 | +0 |

### 4.3 人口トレンド

| 人口傾向 | 点数目安 |
|---|---:|
| 増加 | +2.0 |
| 横ばい | +1.0 |
| 微減 | +0.5 |
| 大幅減少 | +0 |

### 4.4 駅力・流動性

| 条件 | 点数目安 |
|---|---:|
| 乗降客数が多い主要駅 | +1.0 |
| 複数路線利用可能 | +0.5 |
| 人口集中地区内 | +0.5 |
| 賃貸募集が少なすぎる | -0.5 |
| 売買事例がほぼ無い | -1.0 |

### 4.5 未入力時の扱い

立地情報がない場合、いきなり0点にはしません。

MVPでは中立値として3点を置きます。

ただし、データ不足として表示します。

```text
立地スコア: 3/5
注意: 駅徒歩、人口、ハザード、近隣賃料が未取得のため中立評価です。
```

## 5. 融資条件スコア

### 5.1 見る指標

- DSCR
- LTV
- 金利
- 返済期間
- 建物の残存耐用年数
- 自己資金比率

### 5.2 DSCR点数

| DSCR | 点数 | 意味 |
|---:|---:|---|
| 1.50以上 | 5 | 返済余裕が大きい |
| 1.30以上 | 4 | 安全域あり |
| 1.15以上 | 3 | 最低限の余裕あり |
| 1.00以上 | 2 | ぎりぎり返済可能 |
| 0.90以上 | 1 | 危険水準 |
| 0.90未満 | 0 | かなり危険 |

### 5.3 LTV補正

LTVは、物件価格に対する借入割合です。

```text
LTV = 借入額 ÷ 物件価格 × 100
```

| LTV | 評価 |
|---:|---|
| 70%以下 | 安全性が高い |
| 80%以下 | 標準的 |
| 90%以下 | やや高い |
| 100%以下 | フルローンに近い |
| 100%超 | 過剰融資前提。減点 |

補正:

| 条件 | 補正 |
|---|---:|
| LTV100%超 | -1点 |
| LTV95%超 | -0.5点 |
| 自己資金が少ない | 注意表示 |

### 5.4 金利補正

| 金利 | 補正 |
|---:|---:|
| 2%未満 | 補正なし、または軽微な加点 |
| 2〜3% | 標準 |
| 3〜4% | -0.5点 |
| 4%以上 | -1点 |

### 5.5 返済期間補正

返済期間が長いほど毎月の返済は軽くなりますが、出口や建物寿命との整合性が重要です。

| 条件 | 補正 |
|---|---:|
| 返済期間が残存耐用年数と概ね整合 | 補正なし |
| 返済期間が残存耐用年数を大きく超過 | -0.5〜-1点 |
| 築古木造で長期返済前提 | 注意表示 |

## 6. リスク耐性スコア

### 6.1 見る指標

- 損益分岐稼働率
- 家賃下落耐性
- 空室損率
- 修繕リスク
- ハザード
- 金利上昇耐性

### 6.2 損益分岐稼働率

```text
損益分岐稼働率 = 必要収入 ÷ 満室賃料 × 100
```

必要収入には以下を含めます。

- ローン返済
- 管理費
- 修繕費
- 固定資産税
- 保険料

| 損益分岐稼働率 | 点数 | 意味 |
|---:|---:|---|
| 70%以下 | 5 | 空室や家賃下落に強い |
| 80%以下 | 4 | 比較的強い |
| 90%以下 | 3 | 標準 |
| 100%以下 | 2 | 満室なら耐えられる |
| 110%以下 | 1 | 危険 |
| 110%超 | 0 | かなり危険 |

### 6.3 家賃下落耐性

```text
家賃下落耐性 = 100% - 損益分岐稼働率
```

例:

```text
損益分岐稼働率 80%
家賃下落耐性 20%
```

意味:

- 家賃が20%下がっても、理論上は赤字になりにくい
- ただし大規模修繕や長期空室は別途考える必要がある

### 6.4 ハザード減点

| リスク | 減点・ゲート |
|---|---|
| 洪水浸水想定区域 | -0.5〜-1点 |
| 浸水深が大きい | ランク上限CまたはD |
| 土砂災害警戒区域 | -1点以上 |
| 土砂災害特別警戒区域 | ランク上限D候補 |
| 津波浸水想定 | -0.5〜-1点 |
| 大規模盛土造成地 | 注意表示、必要なら減点 |

## 7. 価格妥当性スコア

価格妥当性は、「高値掴みではないか」を見る軸です。

### 7.1 簡易版

MVPの簡易版では、期待利回りと比較します。

```text
価格妥当性比率 = 実際の表面利回り ÷ 期待表面利回り
```

| 価格妥当性比率 | 点数 |
|---:|---:|
| 1.30以上 | 5 |
| 1.15以上 | 4 |
| 1.00以上 | 3 |
| 0.85以上 | 2 |
| 0.70以上 | 1 |
| 0.70未満 | 0 |

期待利回りの初期値:

| 条件 | 期待利回り |
|---|---:|
| RC/SRC・築20年以内 | 7.0% |
| RC/SRC・築20年超 | 8.5% |
| 鉄骨 | 8.5% |
| 木造・築20年以内 | 9.5% |
| 木造・築20年超 | 11.0% |

### 7.2 本番版

本番では、次の3つを組み合わせます。

```text
妥当価格 = 類似取引価格 × 重み + 収益還元価格 × 重み + 積算価格 × 重み
```

#### 類似取引価格

```text
類似取引価格 = 近隣取引事例の中央値単価 × 対象面積
```

見るべき補正:

- 距離
- 築年数
- 構造
- 面積
- 用途地域
- 駅徒歩
- 接道
- ハザード
- 取引時期

#### 収益還元価格

```text
収益還元価格 = NOI ÷ キャップレート
```

キャップレートは、エリア・構造・築年数・リスクで変わります。

#### 積算価格

```text
積算価格 = 土地評価額 + 建物残存価値
```

建物残存価値:

```text
建物残存価値 = 建物面積 × 再調達単価 × 残存価値率
```

### 7.3 価格乖離率

```text
価格乖離率 = 売買価格 ÷ 妥当価格
```

| 価格乖離率 | 判定 | 点数目安 |
|---:|---|---:|
| 0.85以下 | 割安 | 5 |
| 0.85〜0.95 | やや割安 | 4 |
| 0.95〜1.05 | 妥当 | 3 |
| 1.05〜1.15 | やや割高 | 2 |
| 1.15〜1.30 | 割高 | 1 |
| 1.30超 | かなり割高 | 0 |

## 8. 出口戦略スコア

出口戦略は、「最後に売れるか」「売れなくても土地値で守られているか」を見る軸です。

### 8.1 土地値カバー率

```text
土地値カバー率 = 土地評価額 ÷ 売買価格 × 100
```

| 土地値カバー率 | 点数 | 初心者向け意味 |
|---:|---:|---|
| 100%以上 | 5 | 土地値だけで価格をほぼカバー。出口が強い |
| 80%以上 | 4 | 土地値の下支えがある |
| 60%以上 | 3 | ある程度の下支えあり |
| 40%以上 | 2 | 弱めだがゼロではない |
| 20%以上 | 1 | 土地値の下支えはかなり弱い |
| 20%未満 | 0 | 出口リスクが高い |

### 8.2 土地評価額の出し方

#### 路線価方式

```text
土地評価額 = 路線価 × 土地面積 × 補正率
```

必要データ:

- 路線価
- 土地面積
- 奥行
- 間口
- 接道
- 私道負担
- セットバック
- 不整形地補正

#### 倍率方式

```text
土地評価額 = 固定資産税評価額 × 評価倍率
```

必要データ:

- 固定資産税評価額
- 評価倍率

#### 地価公示代替方式

```text
代替土地評価額 = 近傍地価単価 × 土地面積 × 0.8 × 補正率
```

注意:

- これは正確な路線価ではなく参考値
- 正式評価には路線価や固定資産税評価額が必要

### 8.3 残存耐用年数

| 構造 | 法定耐用年数目安 |
|---|---:|
| 木造 | 22年 |
| 軽量鉄骨 | 27年 |
| 鉄骨 | 34年 |
| RC/SRC | 47年 |

```text
残存耐用年数 = 法定耐用年数 - 築年数
```

注意:

- 法定耐用年数は税務上の年数であり、実際の寿命とは違います。
- ただし融資・出口の目安として使われることがあります。

### 8.4 出口戦略の減点例

| 条件 | 減点・ゲート |
|---|---|
| 法定耐用年数超過 | 減点 |
| 土地値カバー率40%未満 | 減点 |
| 再建築不可 | 大幅減点、ランク上限 |
| 接道不良 | 大幅減点 |
| ハザード重大 | 減点、ランク上限 |
| 売買事例が少ない | 流動性リスク |
| 人口減少が強い | 出口リスク |

## 9. 物件タイプ別の見方

### 9.1 戸建て賃貸

重視するもの:

- 土地値カバー率
- 再建築可否
- 接道
- 修繕費
- 賃貸需要
- 売却時に実需層が買えるか

注意:

- 表面利回りが高くても、再建築不可なら出口が弱い
- 土地値が厚いと守りが強い

### 9.2 区分マンション

重視するもの:

- 管理費・修繕積立金
- 修繕積立金残高
- 総戸数
- 管理状況
- 駅徒歩
- 売買事例

注意:

- 土地値カバー率は戸建てより使いにくい
- 管理組合リスクが大きい

### 9.3 一棟アパート

重視するもの:

- 入居率
- レントロール
- 修繕履歴
- 融資条件
- 土地値
- 法定耐用年数

注意:

- 部屋数が多い分、運営力が必要
- 築古木造は融資と出口に注意

### 9.4 一棟マンション

重視するもの:

- NOI
- DSCR
- 大規模修繕
- エレベーター、受水槽、消防設備
- RCの残存耐用年数
- エリア流動性

注意:

- 修繕費の金額が大きくなりやすい
- 表面利回りだけで判断しない

## 10. 初心者向けの最終判断ルール

初心者には、次のようなルールを推奨します。

### 10.1 買ってよい可能性がある条件

- DSCRが1.2以上
- 月間CFがプラス
- 土地値カバー率が60%以上、または出口の根拠がある
- 近隣賃料が入力賃料を支えている
- ハザード重大該当がない
- 修繕履歴が確認できる
- 固定資産税、管理費、保険料を入れてもCFが残る
- 再建築可否が確認できている

### 10.2 初心者は避けた方がよい条件

- DSCRが1.0未満
- 月間CFが赤字
- 満室前提でしか成り立たない
- 想定賃料が相場より明らかに高い
- 再建築不可なのに土地値も薄い
- ハザード重大該当
- 修繕履歴が不明で築古
- 融資期間が建物寿命と合っていない
- 売却事例がほとんどない
- 資料が不足しているのに高ランクに見える

## 11. レポート文言テンプレート

### S

```text
収益性、融資返済、価格妥当性、出口戦略のバランスが良い候補です。
ただし、現地確認、修繕履歴、賃貸需要、法的制限、融資条件の最終確認は必須です。
```

### A

```text
有力候補です。一部に確認すべき点があります。
価格交渉、融資条件、土地値、修繕履歴を確認したうえで検討してください。
```

### B

```text
条件次第で検討可能です。購入価格、融資期間、金利、修繕費の前提を変えると評価が大きく変わる可能性があります。
```

### C

```text
慎重検討が必要です。初心者がそのまま購入するには不安があります。追加調査と価格交渉が必要です。
```

### D

```text
原則見送りです。収支、融資、出口、リスクのいずれかに大きな問題があります。
```

### E

```text
投資対象として危険度が高いです。特殊な再生ノウハウや明確な出口戦略がない限り、見送りを推奨します。
```
